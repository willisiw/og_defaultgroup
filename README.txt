-- SUMMARY --

In a new node that is an OG content type, set the default group.

No Configuration available.  The default group will be set to the most recently
changed OG Group which the user is a member of.


For a full description of the module, visit the project page:
  https://github.com/willisiw/og_defaultgroup

To submit bug reports and feature suggestions, or to track changes:
  https://github.com/willisiw/og_defaultgroup/issues


-- REQUIREMENTS --

* Drupal Organic Groups

-- INSTALLATION --

* Install module using the zip file.  For further info visit 
  http://drupal.org/documentation/install/modules-themes/modules-7

-- CONFIGURATION --

* None.
